# Currency-Converter
Currency Converter Автор: Артур Пименов  Описание: Простое приложение для конвертации валют с сохранением истории операций. Курсы валют загружаются онлайн через внешний API.  Как получить API-ключ:  Перейдите на exchangerate-api.com Зарегистрируйтесь и получите бесплатный ключ Вставьте ключ в программу (в переменную api_key)
